1.4. nh gi m hnh

Ta s nh gi m hnh dng Mean Squared Error (MSE) v R-Squared score ($R^2$).

$$MSE(y, \hat{y}) = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y_i})^2$$

$$R^2 = 1 - \frac{\sum_{i=1}^{n} (y_i - \hat{y_i})^2}{\sum_{i=1}^{n} (y_i - \overline{y})^2}$$

Trong :

  • $\hat{y_i}$ l gi tr m m hnh cho $x_i$

  • $y_i$ l output tht s cho $x_i$

  • $n$ l s lng phn t trong d liu

  • $\overline{y} = \frac{1}{n} \sum_{i=1}^{n} y_i$

# In ra MSE

# Tnh R-squred score trn test data

# In ra R-squred score

2. To nhiu m hnh Linear Regression vi th vin scikit-learn

In[]:
# Khi to mt list  lu cc gi tr MSE cho mi m hnh
mse_train_list = []
mse_test_list = []
coef_list = []
intercept_list = []

# To mt list cc kt hp c th c ca cc thuc tnh  to cc m hnh khc nhau
cols = [['TV'],['radio'],['newspaper'],['TV','radio'],['TV','newspaper'],['radio','newspaper'],['TV','radio','newspaper']]

# Lp qua tt c cc kt hp c th c
for i in cols:
    # Thit lp mi b kt hp trong list trn nh l `X`
    X = ads[i]
    
    # Thit lp "sales" nh l `y`
    y = ads["sales"]
   
    # Tch data thnh train/test sets vi t l 80%/20% 
    # Thit lp random_state = 0 (random_state nh hng n tnh ngu nhin  mi ln chy)
    X_train, X_test, y_train, y_test = train_test_split(X, y, test_size = .2, random_state = 0)

    # Khi to mt Linear Regression model
    lr = LinearRegression()

    # Fit m hnh trn train data
    lr.fit(X_train, y_train)
    
    # Predict trn train v test data dng m hnh  hun luyn
    y_train_pred = lr.predict(X_train)
    y_test_pred = lr.predict(X_test)
    
    # Tnh MSE trn train v test data
    mse_train = mean_squared_error(y_train, y_train_pred)
    mse_test = mean_squared_error(y_test, y_test_pred)
    
    # Thm Train MSE v Test MSE vo kt qu
    mse_train_list.append(mse_train)
    mse_test_list.append(mse_test)
    
    # Thm cc h s ca mi m hnh vo kt qu
    coef_list.append(lr.coef_)
    intercept_list.append(lr.intercept_)

from sklearn.linear_model import Lasso
from sklearn.model_selection import KFold
from sklearn.metrics import mean_squared_error

features = ['crim', 'zn', 'indus', 'chas', 'nox', 'rm', 'age', 'dis', 'rad', 'tax', 'ptratio', 'black', 'lstat']
label = ['medv']

X = boston[features]
y = boston[label]

alphas = (0.001, 0.01, 0.1, 1, 10, 100, 1000)

K = 10
kf = KFold(n_splits = K, shuffle = True, random_state = 1)

for alpha in alphas:
    train_mses = []
    test_mses = []
    for train_index, test_index in kf.split(X, y):
        # split data theo train_index v test_index
        X_train, X_test = X.iloc[train_index], X.iloc[test_index]
        y_train, y_test = y.iloc[train_index], y.iloc[test_index]

        # To model theo alpha
        lasso_reg = Lasso(alpha, fit_intercept = True, random_state = 1)
        
        # Fit the model trn train set
        lasso_reg.fit(X_train, y_train)
        
        # Predict trn train v test set
        y_train_pred = lasso_reg.predict(X_train)
        y_test_pred = lasso_reg.predict(X_test)
        
        # Tnh train v test mse
        train_mse = mean_squared_error(y_train, y_train_pred)
        test_mse = mean_squared_error(y_test, y_test_pred)
        
        # Thm train v test mse vo cc list tng ng
        train_mses.append(train_mse)
        test_mses.append(test_mse)
    
    # In ra train, validation, v test MSE theo alpha
    print('alpha: {:6} | mean(train_mse): {:6} | mean(test_mse): {:6}'.
          format(alpha, round(train_mse, 2), round(test_mse, 2)))
